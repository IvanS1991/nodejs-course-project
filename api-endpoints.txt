Users

* Register
POST - /users/register - { username:..., passHash:... }
* Login
PUT - /users/auth - { username:..., passHash: ... }
* Check own profile
GET - /users/profile - header X-authKey
* Check an user's profile
GET - /users/profile/:id
* Update profile
PUT - /users/update - { stuff to update } - header X-authkey

Collections

* Create
POST - /collections/create - { name:..., private:boolean } - header X-authkey
* View
GET - /collections/view/:id
* Update
PUT - /collections/update/:id - { stuff to update } - header X-authkey
* Delete
DELETE - /collections/delete/:id - header X-authKey

Comments

* Create
POST - /comments/create - { title:..., content:..., movieId:... } - header X-authkey
* Delete
DELETE - /comments/delete/:id - header X-authKey

Movies

* Update
POST - /movies/update
* View one
GET - /movies/view/:id
* View some
GET - /movies/view?page=...&size=...&genres=... - page=2&size=10&genres=action,comedy will select the movies with those two genres and return a slice with the 10th to the 20th from the matches
